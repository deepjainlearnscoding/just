## Quick 3-Step Process:

### Step 1: Push to GitHub (5 minutes)

Open your terminal in the website folder and run:

```bash
# Initialize git repository
git init

# Add all files
git add .

# Commit your files
git commit -m "Valentine website for Nidhi"

# Create repository on GitHub first, then:
git remote add origin https://github.com/YOUR-USERNAME/valentine-nidhi.git

# Push to GitHub
git branch -M main
git push -u origin main
```

### Step 2: Enable GitHub Pages (2 minutes)

1. Go to your repository on GitHub

3. Under "Source": Select **main** branch, **/ (root)** folder
4. Click **Save**

### Step 3: Get Your URL (1 minute)

Your website will be live at:
```
https://YOUR-USERNAME.github.io/valentine-nidhi/
```
